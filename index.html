Data Analysis Dashboard Projects

Figure 2: Power BI Dashboard highlighting road accident patterns and severity analysis.
This dashboard analyzes road accident data to uncover trends based on location, time, and severity. The dashboard supports data-driven policymaking aimed at reducing accident rates and improving road safety.
